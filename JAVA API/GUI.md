## 一、概述

GUI就是图形用户界面

JAVA GUI核心技术：AWT、Swing、SWT(IBM)



---



## 二、AWT

Java最早的GUI工具包

依赖操作系统原生 GUI 组件

功能有限（只提供最基本的组件）



---



## 三、Swing

AWT 的替代品 (JDK 1.2 引入)

完全用 Java 绘制，不依赖原生组件，跨平台一致性

丰富的组件库

内存占用较高，性能略低于 AWT





---



## 四、SWT

Eclipse 开发的 GUI 工具包（不是 Java 标准库的一部分

使用本地组件但提供统一 API

性能接近原生应用
